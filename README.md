## How to Deploy (Streamlit Cloud)
1. Create a GitHub repository and upload all files.
2. Log into [Streamlit Cloud](https://streamlit.io/cloud)
3. Click 'New App' and select:
   - `spiral_portal_pages.py` as the entry file.
4. Click **Deploy**.
